    Introduction

    Welcome to the VoteItUp API! You can use our API to access VoteItUp API endpoints, which can get information about polls and users from our platform.

    VoteItUp is intended to be the reference surveys platform.

    Authentication

    Request

    GET api_endpoint_here HTTP/1.1
Authorization: Bearer <jwt>

    curl "api_endpoint_here"
  -H "Authorization: Bearer <jwt>"

    VoteItUp uses JWT tokens to allow access to the API.

    VoteItUp expects for the API key to be included in all API requests to the server in a header that looks like the following:

    Authorization: Bearer <jwt>

    You must replace <code> with the provided JWT.

    Get Authorization token

    Request

    POST /api/v1/auth/token HTTP/1.1
Content-Type: application/json
Host: voteitup.herokuapp.com

{
    "email": "xyz@mail.com",
    "password": "xyz"
}

    curl "https://voteitup.herokuapp.com/api/v1/auth/token"
  -H "Content-Type: application/json"
  -X POST
  -d '{"email":"xyz@mail.com","password":"xyz"}'

    Response

    {
    "token": "jwt"
}

    To get a JWT you must provide your credentials to authenticate.

    First register the user with a valid email and password.

    Create Poll to a specific User

    Request

    POST /api/v1/users/1/polls HTTP/1.1
Content-Type: application/json
Host: voteitup.herokuapp.com

{
    "text": "#HuelgaAlumnos",
    "choices": [ "Sí", "No" ],
    "area": {
        "city": "Madrid",
        "country": "España"
    }
}

    curl "https://voteitup.herokuapp.com/api/v1/users/1/polls"
  -H "Content-Type: application/json"
  -X POST
  -d '{"text":"#HuelgaAlumnos","choices":["Sí","No"],"area":{"city":"Madrid","country":"España"}}'

    Response

    {
    "status": "success"
}

    Errors

    The VoteItUp API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request is wrong.
    401 Unauthorized -- Your API key is wrong.
    404 Not Found -- The specified route could not be found.
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
